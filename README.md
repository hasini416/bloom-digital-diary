Bloom is a personal digital diary that helps users gently reflect, track habits, moods, and goals over time.

## Features
- Secure user authentication (JWT)
- Daily mood tracking
- Habit creation and daily tracking
- Time-based goal management
- Personal journaling
- Private, user-specific data

## Tech Stack
- Frontend: React (JavaScript)
- Backend: Spring Boot
- Database: PostgreSQL
- Authentication: JWT + Spring Security

## Why Bloom?
Bloom focuses on simplicity, clean architecture, and meaningful data.  
It is designed as a scalable full-stack project to demonstrate real-world engineering skills.

## Future Improvements
- Analytics & charts
- Streak tracking
- Calendar view
